## 操作方法
- V：切換第一人稱與第三人稱視角

### 第三人稱視角操作
- W/S：視角前後移動
- A/D：視角左右移動
- 點擊右鍵：旋轉視角

### 第一人稱視角操作
- 方向鍵（上/下）：無人機前後移動
- 方向鍵（左/右）：無人機左右旋轉
- O/P：無人機向上/下飛行
